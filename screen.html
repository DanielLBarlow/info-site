Commands for screen

    How to install screen on Ubuntu and Debian

  • sudo apt-get install screen

    • How to check the current version of screen

  • screen --version

    • How to make a screen

  • screen

    • How to view the commands available for screen

  • Ctrl + A then ?

    • Press the control button and the letter A at the same time then release these keys and press the ? key

    How to create a screen with a name

  • screen -S {screen_name}

    • Remove the brackets when typing the name of the screen. Screen name can not include spaces

    How to list available screens

  • screen -ls

    • How to detatch from the named screen

  • Ctrl + A then D

    • Press the control button and the letter A at the same time then release these keys and press the letter D

    How to reattach to a screen

  • screen -r {screen_name}

    • Remove the brackets when typing the name of the screen

    You can also use the PID number instead of the screen name reattach

    How to rename a screen

  • Ctrl + a then : sessionname{new_screen_name}

    • Press the control button and the letter A at the same time then release these keys and press the : key

    This command needs to be executed within the screen you want to rename.

    when renaming the screen you must type `sessionname{new_screen_name}`

    How to end a screen session(Attached to the screen)

  • Ctrl + A then K then Y to confirm

    • Press the control button and the letter A at the same time then release these keys and press the letter K.

    A confirmation will pop up saying are you sure. Hit Y to confirm

    How to end a screen session(Not attached to the screen)

  • screen -S {screen_name} -X quit

    • You can also use the PID number instead of the screen name to quit
